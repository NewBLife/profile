# 自己紹介

>## 概要
- 内視鏡、病院検診検査、POS機販売、通信交換機など関連の知識があります
- 9年.NET（WPF、UWP、Xamarin.Froms、WinForm、ASP.NETなど）開発経験
- ソリューション構築と、要件定義から運転保守までの経験をお持ちます
